## 电力设备

### 兆威机电

兆威机电是一家 精密机电控制与自动化系统供应商，主要产品涉及微型电机、微型驱动系统、精密减速器，以及相关的智能装备控制系统。公司业务主要面向 智能制造、新能源、机器人、汽车电子及消费电子等领域。

### 东方电气

东方电气是一家 大型能源装备制造商，业务覆盖火电、核电、风电、燃气发电等全产业链，同时涉足新能源装备和智能制造。

主要业务板块：

火电装备制造

锅炉、汽轮机、发电机组等核心设备

传统业务，但订单稳定，利润贡献长期稳定

新能源装备

核电设备、风电机组、水电设备

随着碳中和、能源转型政策推进，增长潜力较大

燃气与环保设备

燃气轮机、燃烧系统及环保节能设备

在新能源、清洁能源及环保方向具有布局

工程总包及系统集成

电站工程总承包（EPC）

提供从设计、制造到安装的一体化服务

收入结构特点：

传统火电设备仍占核心营收基础

新能源（核电、风电、水电）和燃气设备业务增长快

EPC/系统集成业务有利于提升毛利率和客户粘性

核心优势：

技术实力强，国内发电设备龙头之一

国家能源装备战略的重点企业，政策扶持明显

### 科士达

主营电力电子设备 / 数据中心基础设施 + 新能源 /储能 /光伏 /充电桩。

具体包括：不间断电源 (UPS)、蓄电池 /储能系统、为数据中心提供电力保障与基础设施；此外公司还做光伏逆变器、储能系统、电动汽车充电桩等新能源／储能相关产品。

公司定位为“数据中心基础设施 + 新能源储能”双轮驱动 —— 即在传统 UPS /电源保障基础上，扩展到绿色 /新能源 /储能 /光伏领域。

总结：科士达主要是“电源 + 数据中心基础设施 + 新能源/储能 + 充电 /光伏设备”供应商。它的优势在于电力电子领域积累 + 应对数字化 /新能源浪潮 + 多元化产品线。

### 通威股份

通威是一个“垂直一体化 + 多元”型集团 — 集团业务包括农业（饲料、水产养殖） + 新能源（光伏硅料与电池/组件）两大主线。

在光伏 /新能源领域，它是 上游高纯晶硅 + 中游太阳能电池与组件 的龙头企业：公司拥有从高纯晶硅 → 电池片 → 组件 的完整产业链，并且产能巨大（高纯硅料 + 电池 + 组件产能均处于行业领先位置）

换句话说，通威的核心业务定位在 “硅料 + 电池/组件制造 + 部分光伏系统／电站（含“渔光一体”等）” — 属于光伏产业链 上 & 中游整合。

优势 /意义：由于产业链完整，通威在光伏行业的波动中具有较强的抗风险与自主控制能力；对硅料、电池、组件端都有布局。

### 爱旭股份

爱旭股份主要专注于太阳能电池片和光伏组件（组件 = 把电池片封装成可用太阳能板），以及相关的光伏产品制造。

公司强调技术创新 —— 例如采用单晶硅 / 双面 / N‑型 /高效电池 /高效组件 + 光储一体化方案。

爱旭也在扩产 —— 例如其公告中提到扩大 N‑型 TOPCon 高效晶硅电池产能。

定位总结：爱旭是“太阳能电池片 + 高效组件制造 + 光伏系统 /组件供应 + 技术创新导向”的公司。

### 德业股份

德业股份也属于“逆变器 / 光伏设备 /储能设备”阵营 — 公司主营业务包括逆变器，以及向海外市场出口光伏逆变器与储能／电站相关设备。近年其业务结构与阳光电源、锦浪类似。

根据 2025 年最新数据，德业股份在半年度报告中归母净利润同比增长，海外收入比重增加，这意味着其国际销售／出口是增长重要驱动。

有分析对比指出，在主要逆变器厂商中，德业股份的毛利率 + 净利率整体表现较好。

总结定位：德业股份定位为 光伏逆变器 + 储能／光伏系统设备 + 出口导向 企业，是和阳光电源／锦浪并列的逆变器／储能设备提供者。

### 金风科技

核心业务

全球领先的风力发电整机制造商，业务覆盖陆上风电、海上风电、智能风电运维等。

产品涵盖 1.5MW–6.X MW 陆上风机、7–15MW 海上风机，技术路线成熟。

海外市场广泛，出口至亚洲、欧洲、南美等地区。

优势

技术成熟、风机性能稳定，可靠性高。

市场份额大，行业经验丰富，海外布局强。

持续布局海上风电和智能运维，具备未来增长潜力。

潜在风险

风电补贴政策或市场化电价变化可能影响收益。

上游原材料成本（钢材、稀土永磁材料、叶片材料等）波动影响毛利率。

海上风电项目周期长，资本占用大。

### 国轩高科

核心业务

动力电池研发与制造，主打 磷酸铁锂动力电池。

产品广泛应用于新能源汽车、电动工具及储能系统。

正在拓展海外市场及储能项目。

优势

在磷酸铁锂电池领域经验丰富，技术稳定，成本相对低。

客户群体较广，包括自主车企及部分国际客户。

潜在风险

动力电池行业竞争激烈，尤其与宁德时代、比亚迪相比体量有限。

新能源车市场需求变化、原材料价格波动影响显著。

### 恩捷股份

核心业务

国内领先的 锂电池隔膜制造商。

产品应用于动力电池、储能电池及 3C 消费电子电池。

技术路线涵盖 湿法隔膜（PP/PE）和涂覆隔膜，国内市场份额高。

优势

国内隔膜龙头，技术先进，产品质量稳定。

客户集中度高，与宁德时代、比亚迪等龙头厂商长期合作。

隔膜下游需求增长快，受新能源汽车和储能市场支撑。

潜在风险

客户高度集中，议价能力受限。

原材料（聚合物、涂覆材料）价格波动影响毛利。

技术迭代要求持续研发投入，保持领先优势压力大。

### 特变电工股份有限公司

核心业务

国内领先的 电力设备与输变电解决方案供应商。

产品涵盖 变压器、光伏逆变器、输配电设备、电气成套设备。

同时布局 新能源发电（光伏、风电）及储能。

优势

电力设备龙头，客户覆盖全国电网及新能源企业。

新能源及储能设备研发能力强，整体产业链完整。

具备稳定订单和工程项目背景，抗周期能力强。

潜在风险

电网投资节奏可能受宏观政策和经济影响，短期增长波动。

新能源设备竞争加剧，毛利率可能承压。

### 思源电气

核心业务

主营 智能电气成套设备及电力自动化系统。

产品涵盖高压开关设备、配电柜、储能成套设备及自动化控制系统。

优势

技术积累深厚，智能化和自动化电气系统能力强。

在配电与储能成套设备市场有一定市场份额。

潜在风险

新能源储能系统市场竞争加剧，成长压力大。

下游订单周期长，短期现金流波动明显。

### 正泰电器

核心业务

国内领先的 低压电器、配电设备及光伏逆变器 供应商。

产品涵盖低压断路器、接触器、继电器、配电箱、光伏组件及逆变器。

下游客户涵盖工业、商业、居民及新能源光伏电站。

优势

国内低压电气龙头，品牌知名度高，渠道覆盖全国。

新能源光伏逆变器业务持续扩张，受光伏装机增长驱动。

技术研发和生产能力强，产业链完整。

潜在风险

光伏及低压电器业务受政策及宏观经济周期影响。

产品同质化严重，竞争加剧可能压缩毛利率。

### 四方股份

核心业务

主要从事 电力自动化、智能电网设备及电气自动化成套产品。

产品包括配电自动化设备、变电站成套设备、智能仪表、能源管理系统。

优势

技术经验丰富，产品线较全面。

市场主要面向电网公司，订单相对稳定。

有部分新能源项目经验，可参与风电/光伏配网改造。

潜在风险

行业增长受电网投资周期性限制。

市场竞争加剧，毛利率可能受到压力。

### 万马股份

核心业务 /定位

万马股份是一个比较多元化／升级型电缆企业 — 主业为 电线电缆 /电力电缆，同时也发展 高分子材料 (绝缘料 /护套料 /特种线缆材料) 和 新能源 /智能 /装备线缆 /特种线缆。

公司将传统线缆业务 + “新材料 +新能源／高压／高端 / 特殊用途 /装备线缆 +工业／机器人线缆 /高端线缆” 作为“双引擎 /多元业务”发展战略。

近期 /近期优势

尽管行业整体调整／压价，万马仍“逆势增长”：2024 年营业收入与净利润同比增长，公司披露电缆业务 +高分子材料 +新能源/装备线缆板块都有亮眼表现。

特别是其“高压 /装备 /特种 /新能源 /机器人线缆 +新材料”布局，使其在传统电缆基础上兼具成长性与差异化 — 例如公司成功开发新能源线缆、机器人／装备线缆、光伏／充电桩线缆等新产品。

公司在电网项目 /订单 /新中标项目 /高端客户方面表现活跃 — 说明其“稳 + 变 + 转型”都在有推进。

适合的人群 /投资逻辑

如果你看好“电力／能源 + 新能源 + 高端 /特种 /装备 /新能源线缆 +产业升级 +国产替代 +智能制造 /机器人 /新能源 +多元化” — 万马是一个“基础 + 成长 + 转型 + 多元”型企业。

适合中期/中长线持有 +中等 /适中仓位配置 — 因为其业务既有稳定基础，也有潜在增长弹性 (新材料/高端线缆/新能源/装备/出口/多用途)。

需要注意 /风险

虽然多元，但业务线越多，公司管理、成本、供应链、产品质量和市场竞争越复杂 — 如果管理和执行力不够，可能面临混乱或周期性波动。

原材料 (铜、绝缘材料等) 波动 + 下游需求不稳定 + 新材料 /特种线缆市场推广需要时间 — 短期可能有波动。

整个电缆 /线缆行业如果整体供给/需求失衡 (比如建设放缓 /基础设施投资减少 /新能源电缆替代 /价格压缩)，可能对公司产生冲击。

## 计算机

### 浪潮信息

浪潮信息是中国领先的 服务器 / 数据中心基础设施 /云计算 /大数据 /AI 服务器与存储设备 提供商。

它生产通用服务器、GPU／AI 服务器、存储设备，并向政府、互联网公司、云服务商等客户提供完整 IT 基础设施和解决方案。

公司自视为“算力 + 智算 + 云 + 大数据 + AI 基础设施”的基础提供商，是“IT 基础设施 + 算力提供商 + 数据中心 /云服务配套商”。

### 中科曙光

中科曙光则主要集中在 高性能计算机 / 超算 /高端服务器 /大型存储 /关键数据库系统 /数据中心解决方案。

它同时覆盖从硬件 (服务器、存储、计算节点) → 到中间件 /云 /计算 /数据中心 /算力服务 /解决方案 — 尝试打造“算力生态 + 数据基础设施 + 行业/科研 +云 /大数据服务 +安全/计算服务” 的全链条。

在“超算 /高性能 /关键行业 /政府 /科研 /大数据 /云 + AI /算力服务”领域，中科曙光被视为“国家队 /基础设施提供商 /高端服务器厂商 /高端算力供应商”。

### 浪潮软件

从公开资料来看，浪潮软件曾被列为数字政府、政务云、行业信息化等“数字经济 / 数字政府 / 新基建”相关企业。

### 金证股份

它的主要业务聚焦于金融行业，包括但不限于证券、基金、银行、信托、保险、期货、资管、支付结算机构等。

其产品／服务涵盖证券交易系统、交易结算系统、资管／投资管理系统、基金托管与登记（TA）系统、综合金融后台系统、财富管理系统等「全技术栈 + 全产品线」的金融科技解决方案。

公司对金融机构（券商、基金公司、资管机构、银行等）提供定制化 IT／软件服务与技术支持，是“券商后台 + 资管后台 + 金融云 + 金融数字化转型服务”的重要供应商。

金证目前正处于业绩波动／调整期 — 收入和净利润大幅下滑，但现金流尚有一定支撑。这可能与金融行业客户的 IT 投资节奏／金融监管／金融市场环境有关，也可能与公司自身转型、项目推进／交付节奏变化有关。

### 恒生电子

恒生电子是一家国内领先的金融科技 / 金融 IT 服务提供商，专注为证券、基金、银行、期货、保险资管、信托、私募、企业金融等机构提供资本市场相关的软件系统与技术服务。

2025 年对恒生电子而言，是一个“收入有所下滑，但通过成本控制 + 业务调整 + 核心项目交付”使利润恢复／改善的一年 — 盈利能力有所恢复，但收入端及现金流／应收账款状况仍需关注。

### 云赛智联

公司主营业务包括三大板块：云计算 & 大数据 / 数据中心服务 + 行业解决方案（智慧城市、政企／政务／行业数字化）+ 智能产品／软硬件产品服务。

总体来看，云赛智联最近几年在“云服务／大数据／数据中心 + 行业解决方案／智慧城市 + 智算平台 + 智能产品”上持续发力，整体业务和营收／利润有所恢复和增长。

## 电子

### 斯达半导

它专注于 功率半导体器件（IGBT、SiC、MOSFET 等芯片与模块） 的设计、研发、生产与销售。

公司产品线较为完整：IGBT 模块、电压等级从低压到高压（如 600 V、1200 V 等），模块规格、电流等级／电压等级覆盖广。

应用场景广泛，涵盖新能源汽车主电机控制器、工业控制／电源、变频器、家电变频、光伏／风电、新能源发电、储能、电梯、UPS、电焊机等。

综合来看，斯达半导在“功率半导体 + 功率模块 + IGBT / SiC”这一细分且需求广泛的领域，已经建立了一定技术与市场壁垒 — 这对长期竞争与稳健发展是基础。

### 雅克科技

公司最初主业包括 阻燃剂、橡塑助剂；但近年来它通过收购与业务扩展，转型为较为多元的“电子材料 + 半导体化学材料 + 新材料 + LNG 保温 / 绝热材料 /工程材料 + 化工材料”平台型企业。

在“电子 /半导体材料”板块，公司产品包括 前驱体材料／电子特种气体／封装／电子化学品／光刻胶及配套试剂／半导体用电子粉体材料 等 — 为半导体、显示面板、芯片制造等提供基础材料供应。

换句话说，雅克科技正尝试把自己从传统化工／材料 — 发展为“半导体 & 新材料 + 工程材料 + 电子化学品”混合型材料供应商／平台。

换句话说，雅克科技正尝试把自己从传统化工／材料 — 发展为“半导体 & 新材料 + 工程材料 + 电子化学品”混合型材料供应商／平台。

### 紫光国微

紫光国微是一家国内领先的 集成电路（IC）设计与安全芯片／特种芯片 公司，其产品广泛应用于金融支付、SIM / 智能卡、身份证／政务卡、物联网 / 通信 /汽车 /工业等多个领域。

紫光国微近年来经过波动。2023 年营收与利润尚稳；2024 年整体受行业周期与市场因素影响有所下滑；2025 年公司通过产品结构调整、自主芯片 / 安全芯片推广、新市场开拓等方式，业绩有所恢复，盈利能力稳中有进。

### 通富微电
