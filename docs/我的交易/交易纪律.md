# 交易纪律

## 交易原则

风险可控是第一准则。

没有机会可以等，不要强行交易。

错过机会没关系，市场有的是机会。

不要浮盈大幅加仓赌徒心理，浮亏大幅加仓降低成本。

满仓不能超过 3 天。

## 止损

单笔大于1%~2%止损

账户总亏损大于5%止损
